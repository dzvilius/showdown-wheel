## Showdown Wheel

Online slots game built with Pixi.JS

[Live Preview](https://showdown-wheel.pages.dev/)

### Quick Start

```bash
git clone git@github.com:dzvilius/showdown-wheel.git
npm install
npm start
npm run build
```
